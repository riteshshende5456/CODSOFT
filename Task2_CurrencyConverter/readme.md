# 💱 Currency Converter – Java Program

## 📌 About
This project is a **Currency Converter** developed in Java.  
The program allows users to convert one currency into another using predefined conversion rates. It is a **menu-driven console application** that demonstrates arithmetic calculations, user input handling, and decision-making logic.

---

## 🎯 Objective
- Understand arithmetic calculations in Java
- Learn menu-driven program design
- Practice conditional statements
- Build logical thinking skills
- Implement real-world utility program

---

## ⚙️ Features
- Menu-driven interface
- Supports multiple currency conversions
- Uses predefined exchange rates
- Accepts user input amount
- Displays converted result instantly
- Repeats until user exits

---

## 🛠️ Technologies Used
- Java (Core Java)
- Scanner Class (for input)
- Conditional Statements (if/switch)
- Loops (for menu repetition)
- Arithmetic Operations

---

## 🧠 Algorithm Used
**Arithmetic Conversion Algorithm**
1. Start program
2. Display conversion menu
3. Take user choice
4. Input amount to convert
5. Apply conversion formula  
   `Converted Amount = Amount × Rate`
6. Display result
7. Ask user if they want to continue
8. Repeat until exit selected
9. End program

---

## ▶ How to Run Program

Compile:
```bash
javac CurrencyConverter.java
```

Run:
```bash
java CurrencyConverter
```
