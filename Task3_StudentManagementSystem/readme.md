# 🎓 Student Management System – Java Program

## 📌 About
This project is a **Student Management System** developed in Java.  
It is a console-based application that allows users to manage student records efficiently. The system performs operations like adding, viewing, searching, updating, and deleting student details. It demonstrates CRUD operations and searching logic using basic Java concepts.

---

## 🎯 Objective
- Learn how record management systems work
- Understand CRUD operations implementation
- Practice data handling using arrays or lists
- Improve logical thinking and problem-solving skills
- Build a real-world console application

---

## ⚙️ Features
- Add new student record
- Display all student records
- Search student by ID or name
- Update student details
- Delete student record
- Menu-driven interface
- Continuous execution until exit option selected

---

## 🛠️ Technologies Used
- Java (Core Java)
- Scanner Class (for user input)
- Arrays / ArrayList (for data storage)
- Conditional Statements
- Loops
- Functions / Methods

---

## 🧠 Algorithm Used
**CRUD-Based + Linear Search Algorithm**

**Step-by-Step Logic:**
1. Start program
2. Display menu options
3. Accept user choice
4. Perform operation based on choice:
   - Add → store student details
   - View → display all records
   - Search → use linear search to find student
   - Update → modify existing record
   - Delete → remove record
5. Repeat menu until user selects Exit
6. End program

---

## ▶ How to Run Program

Compile:
```bash
javac StudentManagementSystem.java
```

Run:
```bash
java StudentManagementSystem
```
