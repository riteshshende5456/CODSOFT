# 🎯 Number Game – Java Program

## 📌 About
This project is a simple **Number Guessing Game** developed in Java as part of my Java Programming practice.  
The program generates a random number between **1 and 100**, and the user has to guess it. The program provides hints if the guess is too high or too low and also checks if the input is within the valid range.

---

## 🎯 Objective
- Practice core Java programming concepts
- Understand random number generation
- Implement conditional logic
- Learn user input handling
- Build logical thinking skills

---

## ⚙️ Features
- Generates random number between **1 to 100**
- Validates user input range
- Gives hints:
  - Too High
  - Too Low
- Counts number of attempts
- Displays success message when guessed correctly

---

## 🛠️ Technologies Used
- Java (Core Java)
- Scanner Class (for input)
- Random Class (for number generation)
- Conditional Statements
- Loops

---

## 🧠 Algorithm Used
**Guessing Logic Algorithm**
1. Start program
2. Generate random number between 1–100
3. Ask user to guess number
4. If input < 1 or > 100 → show range error
5. If guess > number → print "Too High"
6. If guess < number → print "Too Low"
7. Repeat until guess is correct
8. Display attempts count
9. End program

---

## ▶ How to Run Program

Compile:
```bash
javac NumberGame.java
```

Run:
```bash
java NumberGame
```
